# Tris
Un'app web minimalista per giocare a Tris — un solo file HTML, nessuna dipendenza, nessuna pubblicità, nessuna distrazione.
Nata come esperimento di design: quanto velocemente si può passare da un'idea a un prodotto giocabile e rifinito, collaborando con un'AI?
Il primo prototipo funzionante in 4 minuti. Il resto del tempo speso come ogni designer minimalista paranoico farebbe — affinando, semplificando, togliendo.

# Funzionalità
Layout pulito e responsive
Modalità chiaro e scuro
5 palette di colori (incluse Cyberpunk e Neon Dusk)
Animazioni sul disegno di X e O
Un solo file HTML — basta aprirlo nel browser

# Costruito con
HTML · CSS · Vanilla JS · Claude (Anthropic)
